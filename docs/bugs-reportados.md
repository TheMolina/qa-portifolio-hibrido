# Relatório de Bugs – SauceDemo

## BUG-001 – Mensagem de erro pouco clara no login

**Ambiente:** Chrome / Windows 10  
**Pré-condição:** Usuário na tela de login  

**Passos para reproduzir:**
1. Informar username válido
2. Não informar senha
3. Clicar em Login

**Resultado Atual:** Mensagem genérica de erro  
**Resultado Esperado:** Mensagem indicando campo obrigatório  

**Severidade:** Média  
**Prioridade:** Média
