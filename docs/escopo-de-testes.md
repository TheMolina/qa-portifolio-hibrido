# Escopo de Testes – SauceDemo

## Objetivo
Definir o escopo dos testes a serem executados na aplicação SauceDemo, garantindo a validação das principais funcionalidades do sistema.

## Funcionalidades Dentro do Escopo
- Autenticação de usuário (login)
- Listagem de produtos
- Ordenação de produtos
- Adição e remoção de itens do carrinho
- Fluxo de checkout

## Funcionalidades Fora do Escopo
- Integrações com meios de pagamento reais
- Testes de performance
- Testes de segurança avançados
- Compatibilidade entre navegadores

## Tipos de Teste Aplicáveis
- Testes funcionais
- Testes manuais
- Testes exploratórios
- Testes de regressão
